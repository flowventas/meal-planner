# Planificador semanal de comidas

Aplicacion web estatica para generar un menu semanal saludable, ajustar por numero de adultos, ninos y bebes, y crear una lista de super con costo estimado.

## Funciones

- Genera desayunos, comidas y cenas para toda la semana.
- Permite cambiar una comida individual sin rehacer todo el menu.
- Ajusta cantidades por adultos, ninos y bebes.
- Convierte ingredientes a unidades de compra como kg, litros, docenas, latas y paquetes.
- Calcula un costo semanal aproximado por ubicacion.

## Publicacion

Este proyecto es estatico. En Vercel se puede publicar sin build command; el archivo principal es `index.html`.
